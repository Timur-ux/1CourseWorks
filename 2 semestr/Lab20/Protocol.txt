wc test1 - Печатает количество строк, слов и байтов в файле
wc -с test1 - Печатает только количество байтов
wc -w test1 - Печатает только количество слов
wc -l test1 - Печатает только количество строк
wc -m test1 - Печатает только количество символов
dd if=20/test1 of=20/test2 - Копирует файл в директории 20 test1 в test2 в той же директории
diff test1 file3 - Построчно сравнивает 2 файла
diff -q test1 file3 - Печатает только различаются ли 2 файла
grep -F sd test1 - Ищет и печатает строки файла, в которых содержится паттерн sd, флаг -F показывает, что паттерн - фиксированная строка
join file1 file2 - Соединяет 2 файла по общему полю, в данном случае - по номеру строки, самому первому полю
sort file1 - Печатает в стандартный вывод отсортированный по строкам файл
tail file1 - Печатает последние 10 строк файла
tail -n 3 file1 - Печатает поледние 3 строки файла
tee file2 - Записывает ст. ввод в файл, одновременно выводит его в ст. вывод
tee file2 - Добавляет(не перезаписывает) ст. ввод в файл, одновременно выводит его в ст. вывод
tr a b - Заменяет символы a в ст. вводе на b и выводит полученный результат в ст. вывод
uniq file1 - Выводит уникальные строки в ст. вывод
uniq file1 file2 - Записывает уникальные строки первого файла во второй файл
uniq -D file1 - Выводит только повторяющиеся строки
uniq -d file1 - Выводит только повторяющиеся строки по одному разу
uniq --all-repeated=separate file1 - Выводит только повторяющиеся строки, блоки одинаковых строк разделены пустой строкой
uniq --group file1 - Выводит все строки в ст. вывод, блоки одинаковых строк разделены пустой строкой
od -b file1 - Выводит восьмеричные коды каждого символа из файла, начало файла 0000000, конец число символов в файле, также в восьмеричном формате(смещение)
od -с file1 - Выводит все символы файла, разделенные пробелом
od -An -c file1 - Выводит все символы файла, разделенные пробелом без смещения
od -i file1 - Выводит содержимое файла в виде десятичного целого числа
od -o file1 - Выводит содержимое файла в виде восьмеричных двухбайтовых блоков
sum file1 - Выводит контрольную сумму файла и количество блоков
cut -b 5-20 file4 - Вырезает из файла все байты от 5 до 20 и выводит их в ст. вывод
tar -cvf arch.tar file1 file2 file3 test1 - Создаёт архив arch.tar, архивирует в него 4 файла, и выводит их имя в ст. вывод
tar -tf arch.tar - Выводит содержимое архива в ст. вывод
tar -xvf arch.tar - Разархивирует архив в текущую директорию
gzip -c arch.tar > arch.gz - сжимает архив, результат записывает в arch.gz
gzip -cd arch.gz - разжимает архив обратно, выводит результат в ст. вывод
bzip2 arch.tar - сжимает архив в архив.bz2, сам архив удаляется.
bzip2 -k arch.tar - сжатие с сохранением оригинала
head file1 - Выводит первые 10 строк файла
head -n 2 file1 - Выводит первые 2 строки файла
iconv -f KOI8-RU -t UTF-8 file1 - Переконвертирует файл из KOI8-RU в UTF-8, результат выведет в консоль
iconv -f KOI8-RU -t UTF-8 file1 -o file2 - Переконвертирует файл из KOI8-RU в UTF-8, результат запишет во второй файл
md5sum file2 - Выводит контрольную сумму в ст. вывод
md5sum file2 > file2.md5 - Записывает контрольную сумму в .md5 файл
du Labs - Выводит в ст. вывод то, сколько места занимает данная директория и её поддиректории
du -a Labs - Выводит в ст. вывод то, сколько места занимает каждый файл в директории
du -B K Labs - Выводит в ст. вывод то, сколько места занимает данная директория и её поддиректории в килобайтах
du -c Labs - Выводит в ст. вывод то, сколько места занимает данная директория и её поддиректории и общий занимаемый объём
du -d 1 Labs - Выводит в ст. вывод то, сколько места занимает данная директория и её поддиректории, с глубиной прохода 1
du -h Labs - Выводит в ст. вывод то, сколько места занимает данная директория и её поддиректории, в человеко-читаемом варианте
file file1 - Выводит в ст. вывод тип данных, которые храняться внутри file1
file -b file1 - Выводит в ст. вывод тип данных, которые храняться внутри file1, но не выводит название файла
file -i file1 - Выводит в ст. вывод MIME-тип файла по его заголовку
file -l file1 - Выводит в ст. вывод паттерны файла и их длину.
touch emptyfile - Создает пустой файл
find . -name "file1" - Ищет file1 в текущей директории
find - Ищет все файлы в текущей директории(аналог find . и find . -print)
find -type d - Ищет все папки в текущей директории
find -type f - Ищет все файлы в текущей директории
mktemp tempfileXXX - создает временный файл в текущей директории с именем tempfile{Произвольный набор 4 символов}
mktemp -d tempdirXXX - создает временный каталог в текущей директории
mktemp tempfileXXX --suffix=Alaa - создает в текущей директории временный файл с именем tempfileAlaa
